Home lock using IOT

Developed a face recognition system using IOT for home lock

In our Under graduate final year project we've used technologies like IOT and programming languages python to develop this face recognition system to detect your face based on a pre-given photos and open the door if the face is detected and to send a notification to the owner if the face is not detected.
